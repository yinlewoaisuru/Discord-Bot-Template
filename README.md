```bash
git clone https://github.com/yinlewoaisuru/Discord-Bot-Template.git
```

**Bước 2** — Tải tất cả dependencies:
```bash
npm install
```

**Bước 3** — Tạo file cấu hình:
```bash
cp .env.example .env
```

**Bước 4** — Điền thông tin vào `.env`:
```env
# ── Discord ─────────────────────────────────
TOKEN=your_discord_bot_token_here
CLIENT_ID=your_client_id_here

# ── Database ─────────────────────────────────
MONGO_URI=mongodb://localhost:27017/novela

# ── API Server ────────────────────────────────
API_SECRET=your_strong_secret_here
API_PORT=3000

# ── Environment ───────────────────────────────
NODE_ENV=production
```

**Bước 5** — Khởi động:
```bash
# Production (cluster + sharding)
npm start

# Development (auto-restart khi sửa file)
npm run dev

# Single process (không cluster, test nhanh)
npm run main
```

### 🎨 Embed Color System

All embeds use a **single source of truth** at `bot.config.js → embed.colors`. Zero hardcoded colors anywhere.

```js
// ✅ Correct — always use createEmbed()
const { createEmbed, createErrorEmbed } = require('./utils/embedBuilder');

createEmbed({ type: 'moderation', title: '🔨 Banned', description: '...' });
createEmbed({ type: 'success',    title: '✅ Done',   description: '...' });
createEmbed({ type: 'error',      title: '❌ Error',  description: '...' });

// ❌ Wrong — never hardcode colors
new EmbedBuilder().setColor('#FF0000');
```

### 🏗️ Architecture Patterns

| Pattern | Description |
|:---|:---|
| **Domain-Driven Design** | Business logic separated strictly by domain |
| **Repository Pattern** | Data access fully decoupled from commands |
| **Middleware Pipeline** | `Auth → Cooldown → Permission → Execute` |
| **Event-Driven Architecture** | All interactions via internal EventBus |
| **Dependency Injection** | Manual ServiceContainer |
| **Graceful Degradation** | Fallback for every feature, no full crashes |
